# IxUI

A modern web application for managing IPTV/streaming infrastructure.

- **Backend:** Python/FastAPI — see [`app/backend/`](app/backend/)
- **Frontend:** Lit web components with Tailwind CSS — see [`app/frontend/`](app/frontend/)
- **Legacy:** Original GWT application in [`GWT_APP/`](GWT_APP/)
- **Migration guide:** [`GWT_APP_MIGRATION_GUIDE.md`](GWT_APP_MIGRATION_GUIDE.md)

See [`app/README.md`](app/README.md) for setup and usage instructions.
